Some scripts require MNIST and Facial Expression Recognition data.

- you can get MNIST dataset from:
	 https://www.kaggle.com/c/digit-recognizer;
- download it and change the 'mnist_data' variable value in 'util.py' with the corresponding filepath

- you can get Facial Expression Recognition dataset from:
	 https://www.kaggle.com/c/challenges-in-representation-learning-facial-expression-recognition-challenge/data;
- download it and change the 'fer_data' variable value in 'util.py' with the corresponding filepath
